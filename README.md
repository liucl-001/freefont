# freefont

站酷字体：
开放商业使用，但不能改动
https://www.zcool.com.cn/special/zcoolfonts/

思源字体:
google fonts 的一部分

SIL,不修改，不单独售卖就可以，相当好的一个字体
https://github.com/google/fonts/
